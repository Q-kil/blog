# Install
``` zsh
yarn add prettier --dev --exact
```

# Option
| Name | API Override  | Default | Describe |
| :--- | :--- | 
| Tab Width | tabWidth: <int> | 2 | 额外的空格并不会造成功能性的问题，空格没有必要存在。 |
| singleQuote| singleQuote: <boolean> | false | 单引号 |
